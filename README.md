A custom **Rate Limiter** implementation built with Spring Boot. 
This project is designed to demonstrate multiple rate-limiting strategies, such as Fixed Window, Sliding Window, Token Bucket, and more. 
Features (Planned)
* [x] Basic Spring Boot setup with REST controller 
* [x] Fixed Window rate limiting 
* [x] Sliding Window rate limiting
* [ ] Token Bucket rate limiting
* [ ] Leaky Bucket rate limiting 
*  [ ] Client-specific usage statistics endpoint (/api/rate-limiter/stats)
